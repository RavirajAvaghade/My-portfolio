## ✏️ Customisation Guide

### 🖼 Change Profile Photo
Replace `images/profile.jpg` with your photo (square recommended).

### 📄 Change Resume PDF
Replace `assets/resume/Raviraj_Avaghade_Resume.pdf` with your updated resume.

### 📬 Change Contact Information
In `index.html`, search for `raviraj@example.com` and `+91 00000 00000` and replace.

### 🔗 Change Social Media Links
In `index.html`, search for `href="#"` next to LinkedIn / GitHub icons and add your URLs.

### ➕ Add New Skills
Copy any `.skill-item` block in the Skills section and update the logo src, name, and `--pct` value.

### ➕ Add New Projects
Copy any `.project-card` block in the Projects section and update title, description, tags, and links.
